# Ads Monitoring

## Расписание запуска

Проект рассчитан на запуск по расписанию через `cron`. Добавьте запись в crontab (пример запуска каждый час):

```cron
0 * * * * /usr/bin/env bash -lc 'cd /path/to/ads_monitoring && ./run.sh'
```

> Замените `/path/to/ads_monitoring` и команду `./run.sh` на актуальные для вашей среды.

### Последовательность выполнения

Один запуск должен выполняться строго последовательно:

1. **fetch** — получить данные.
2. **write sheets** — записать данные в Google Sheets.
3. **compare** — сравнить с предыдущими значениями.
4. **notify** — отправить уведомления.

Убедитесь, что ваша командная точка входа (например, `run.sh`) выполняет эти шаги именно в указанном порядке и не запускает следующий шаг до завершения предыдущего.
